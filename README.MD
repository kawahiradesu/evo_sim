# EVO_SIM
環境圧から進化が創発する人工生命シミュレーション

## 概要・目的
「生物を作る」のではなく「進化圧を設計して生命を創発させる」ことを目指して作った個人プロジェクト。完全にランダムな、あったかもしれない地球の生態系を生成したい。現実と似たような生態系が創発した時、その合理性を感じられるのが面白い。設計と実装の練習も兼ねている。

## 主な機能・実装内容
- Perlin Noiseによる自然な地形生成（高度→河川→水分の因果連鎖）
- 季節変動・気候フェーズによる環境の動的変化
- α/βケラチン系統制約によるルートの分岐
- 腸内細菌モデルによる草食行動の創発
- 遺伝子の掛け算から派生形質が生まれるEmergent Trait設計

## 技術スタック
Python / Flask / Flask-SocketIO / NumPy / Matplotlib / Numba

## 実行方法
pip install -r requirements.txt
cd src/
python app.py
起動後、ブラウザで `http://localhost:5000` にアクセス。
